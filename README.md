# steppermotor-controller

### Start Up Instructions
The server needs to be started before the client starts or else the client will have nothing to connect to and will fail.

To start up the server run the following command - <br/>
`python server.py` <br/>
If you receive an error that the `python` command doesnt exist, you could then try - <br/>
`python3 server.py`

Open up another terminal instance and run the following command to start up the client - <br/>
`python client.py` or `python3 client.py`

## Start Up Virtual Python Env (venv)
1. Open up steppermotor folder (project folder)
2. Open steppermotor_venv folder
3. Run the following command - <br/>
`source bin/activate`
